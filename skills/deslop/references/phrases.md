# Phrases to cut or replace

Dense catalog of phrases that almost always signal AI generation. Cut them or replace with something specific.

## Throat-clearing openers

Drop entirely — the reader doesn't need a runway.

- "Here's the thing:"
- "Here's what's interesting:"
- "Here's what you need to know:"
- "Here's the reality:"
- "Let me start by saying…"
- "Before we dive in…"
- "Let's unpack this."
- "Let me explain."
- "To put it simply,"

## Emphasis crutches

Delete. The sentence before should carry its own weight.

- "Let that sink in."
- "Read that again."
- "Think about that for a moment."
- "That's the key takeaway."
- "That matters."
- "This is critical."

## Meta-commentary

The reader already knows what section they're in.

- "In this section, we'll explore…"
- "The rest of this article will discuss…"
- "As we discussed above…"
- "As mentioned earlier,"
- "It's important to note that…"
- "It's worth noting that…"
- "It bears mentioning that…"

## Business jargon

Replace with concrete actions or terms.

| Avoid | Use |
|---|---|
| leverage | use |
| navigate the landscape | (name the specific obstacle) |
| the ecosystem | (name the specific system) |
| synergy | (name what the two things actually produce together) |
| seamlessly integrate | works with |
| robust solution | (name the mechanism) |
| best-in-class | (name the metric) |
| at scale | (name the scale) |
| holistic approach | (name what you include) |
| paradigm shift | (name what changed) |
| unlock value | (name the value) |
| game-changer | (name what it changes) |
| next-generation | (name the version) |
| end-to-end | (name the start and end) |
| actionable insights | (name the action) |

## Adverb crutches

Adverbs ending in `-ly` doing work that a stronger verb or a specific fact would do better.

- quietly (→ "The system quietly dropped…" → "The system dropped … without a log entry.")
- seamlessly
- effortlessly
- fundamentally
- dramatically
- significantly
- notably
- particularly
- undoubtedly
- surprisingly
- interestingly

## AI vocabulary tells

Unmistakable AI word choices. Strip them.

- delve / delving / delved
- tapestry
- realm
- landscape (as metaphor)
- nuanced
- multifaceted
- intricate
- profound / profoundly
- tapestry
- testament (as in "a testament to")
- journey (metaphorical)
- foster / fostered
- empower / empowering
- catalyst
- pivotal

## Vague declaratives

Replace with the specific thing.

| Avoid | Replace with |
|---|---|
| "The reasons are structural." | Name the structural reason. |
| "The implications are significant." | Name the implication. |
| "This has broad applications." | Name two applications. |
| "Performance considerations matter." | Quote the metric. |
| "There are several factors." | List the factors. |
| "Best practices suggest…" | Cite the practice and source. |
| "Experts argue…" | Name one expert. |

## Softeners that add nothing

Cut.

- "It might be worth considering…"
- "You may want to think about…"
- "Perhaps it would be useful to…"
- "Somewhat"
- "Arguably"
- "Relatively"
- "Fairly"
- "Pretty much"

## Hedges disguised as humility

- "Just a quick thought…"
- "Just throwing this out there…"
- "Correct me if I'm wrong, but…"
- "Not sure if this is right, but…"

If you're not sure, say what specifically you're not sure about.

## Signposted conclusions

- "In conclusion,"
- "To wrap up,"
- "To summarize,"
- "All in all,"
- "At the end of the day,"
- "Ultimately,"

If you reach the end with a sentence that re-states the argument, cut the sentence.

## "Despite these challenges…"

Any sentence starting with a transition away from a negative to restate the positive. Rewrite with the specific outcome instead.
