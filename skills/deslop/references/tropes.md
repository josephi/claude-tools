# AI writing tropes — full catalog

All the tells, grouped by category.

## Word choice

| Trope | Why it's a tell | Fix |
|---|---|---|
| "quietly", "silently", "subtly" | Overused adverbs that rarely describe what actually happened | Cut or replace with a specific verb |
| "delve / delving" | One of the strongest AI words | Use "look at", "examine", or nothing |
| "tapestry" | Essentially never used in real prose | Replace with what you mean (a set, a pattern, a mix) |
| "realm" | Pseudo-literary | "area", "field", or cut |
| "landscape" (metaphorical) | Business-jargon AI staple | Name the specific thing |
| "nuanced" | Says nothing concrete | Name the specific nuance |
| "multifaceted" | Same as nuanced | Name the facets |
| "intricate" | Decorative, not descriptive | Describe what makes it complex |
| "seamlessly" | Almost never true | "Works with" or name the seam that isn't there |
| "pivotal" | Overclaims | "important" or name what changed |
| "foster" / "fostered" | Consulting speak | "create", "build", "support" |
| "empower" | Marketing speak | Name the capability you added |
| "catalyst" | Pseudo-technical metaphor | Name the mechanism |
| "testament (to)" | "a testament to the team's dedication" | Name the specific outcome |
| "journey" | Corporate metaphor | Describe the actual sequence |
| "ecosystem" | Overused | Name the system |

## Sentence structure

| Trope | Fix |
|---|---|
| Binary contrast ("Not X. Y.") | State Y directly |
| Negative listing ("Not a X. Not a Y. A Z.") | State Z directly |
| Dramatic fragmentation ("Speed. That's it.") | One sentence |
| Self-posed rhetorical questions | Fold into statement |
| "The X? Y." inversions | Declarative order |
| Serial participle phrases | Cut or make specific |
| Stacked short sentences for emphasis | Vary length |
| Three-word intros ("More importantly,", "Interestingly,") | Cut |

## Paragraph structure

| Trope | Fix |
|---|---|
| Anaphora (3 sentences starting the same) | Vary openings |
| Tricolon (3-item list) everywhere | Use 2 items or 1 |
| "Not A, not B, but C" pattern | State C |
| Paragraph ending with punchy one-liner, repeatedly | Vary |
| Listicle disguised as prose | Use an actual list or one sentence |

## Tone

| Trope | Fix |
|---|---|
| Narrator-from-a-distance ("It has long been recognized that…") | Name who recognized it |
| Pedagogical voice ("Let's break this down.") | Skip the setup |
| False vulnerability ("I know this sounds controversial, but…") | State the claim |
| Grandiose stakes inflation ("This will change everything.") | Name the specific change |
| Patronizing analogies for simple ideas | Trust the reader |
| "It's important to note…" | Delete; note it directly |

## Formatting

| Trope | Fix |
|---|---|
| Em dashes in prose | Comma, period, or parenthesis |
| Bold-first bullets (every item) | Remove the bold leads |
| Unicode arrows in prose | "to" or "becomes" |
| Signposted conclusions ("In conclusion,") | Cut; let the last sentence be the conclusion |
| "Despite these challenges…" | Name the specific outcome |
| Header casing mixed mid-document | Pick one |
| Bullet lists of 1 item | Use prose |

## Composition

| Trope | Fix |
|---|---|
| Fractal summary (say, say, summarize) | Say once |
| Restating the thesis every paragraph | State once, trust the reader |
| Invented concept labels ("the supervision paradox") | Describe the phenomenon instead of naming it |
| Stacked historical analogies ("Apple didn't build Uber. Facebook didn't build…") | Pick one if needed |
| False ranges ("from X to Y") where the range is meaningless | Name the actual spread |
| Generic attribution ("Experts argue…") | Name one expert or cut the claim |

## Dead-giveaway openings and closings

These almost never appear in human-written prose. When they appear in yours, delete them.

- "In today's fast-paced world,"
- "In the ever-evolving landscape of…"
- "As we move into the future,"
- "In conclusion, we have seen that…"
- "Ultimately, the choice is yours."
- "The possibilities are endless."
- "Only time will tell."
- "The future is bright."
